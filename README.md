
# nms_broker

This is a Supervisor Skeleton Application.


----------

补充说明：   
框架最初为了网管（nms）业务所开发，主要负责数据的聚合、处理，和分发；   
框架设计符合 Erlang/OTP 原则，用于创建动态、可定制、具有分布式功能的监督结构；   
框架中对 rabbitmq 客户端，redis 客户端，mysql 客户端进行了集成，后续可以方便的扩展接入其他类型的客户端；
